# Toxic Comment Blocker

This project is a browser extension and Flask backend designed to intercept and block toxic comments on social media platforms like Instagram and YouTube.

## Features
- Real-time toxicity classification.
- Browser extension for seamless integration.
- Flask backend for model inference.
- Dashboard for viewing blocked comments.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the Flask server:
   ```bash
   python app.py
   ```
3. Load the extension in your browser.

## Technologies
- Python (Flask)
- JavaScript (Browser Extension)
- Machine Learning (Toxicity Classification)
